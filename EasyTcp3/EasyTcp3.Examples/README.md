# Install EasyTcp
> dotnet add package EasyTcp <br/>
> dotnet add package EasyTcp.Actions<br/>
> dotnet add package EasyTcp.Encryption<br/>

# EasyTcp.Examples

All examples are well commented and explain what is happening. <br/>
